# 📚 Exercícios de Lógica de Programação -- Portugol

Repositório contendo uma coleção de exercícios desenvolvidos em
**Portugol Studio** com o objetivo de praticar os principais conceitos
de **lógica de programação**.

Os exercícios foram organizados em diferentes trabalhos, cada um
abordando um conceito fundamental da programação como **desvio
condicional, operadores lógicos, laços de repetição, funções,
recursividade, vetores e matrizes**.

------------------------------------------------------------------------

# 📌 Tecnologias Utilizadas

-   Portugol Studio
-   Lógica de Programação
-   Algoritmos estruturados

------------------------------------------------------------------------

# 🧠 Conteúdo dos Exercícios

## 📍 Trabalho 01 -- Desvio Condicional

Exercícios focados no uso de **estruturas condicionais (se, senao)**.

1.  Verificar se um número é **par ou ímpar**
2.  Verificar se uma pessoa é **maior de idade**
3.  **Comparar dois números**
4.  Verificar se um aluno foi **aprovado, recuperação ou reprovado**
5.  **Cálculo de desconto** em compras

------------------------------------------------------------------------

## 📍 Trabalho 02 -- Operadores Lógicos

Exercícios utilizando **E, OU e NÃO**.

1.  Verificar se um número está **entre 10 e 50**
2.  Verificar se uma pessoa pode **se cadastrar**
3.  Verificar se um aluno pode fazer **prova final**
4.  Verificar se um número **não é positivo**
5.  Sistema simples de **login**

------------------------------------------------------------------------

## 📍 Trabalho 03 -- Laços de Repetição

Exercícios utilizando **laços de repetição (para, enquanto,
faça-enquanto)**.

1.  Contagem de **1 até 10**
2.  **Somar números de 1 até N**
3.  **Contagem regressiva** de 10 até 0
4.  **Tabuada** de um número
5.  **Sistema de senha** até acertar

------------------------------------------------------------------------

## 📍 Trabalho 04 -- Funções

Prática de **criação e utilização de funções**.

1.  Função para **somar dois números**
2.  Função para calcular o **quadrado de um número**
3.  Função para verificar se um número é **par ou ímpar**
4.  Função para calcular **média de três notas**
5.  Conversão de **Celsius para Fahrenheit**

Fórmula utilizada:

    Fahrenheit = (Celsius * 1.8) + 32

------------------------------------------------------------------------

## 📍 Trabalho 05 -- Recursividade

Exercícios utilizando **funções recursivas**.

1.  **Fatorial de um número**
2.  **Soma de 1 até N**
3.  **Sequência de Fibonacci**
4.  **Contagem regressiva**
5.  **Inverter uma string**

------------------------------------------------------------------------

## 📍 Trabalho 06 -- Vetores

Manipulação de **vetores (arrays)**.

1.  Armazenar e exibir números
2.  Somar elementos de um vetor
3.  Encontrar o **maior valor**
4.  Contar **números pares**
5.  **Inverter vetor**

------------------------------------------------------------------------

## 📍 Trabalho 07 -- Matrizes

Exercícios utilizando **matrizes (arrays bidimensionais)**.

1.  Criar e exibir uma **matriz 3x3**
2.  **Somar elementos** de uma matriz 5x5
3.  Encontrar o **maior valor**
4.  Somar elementos da **diagonal principal**
5.  **Multiplicar matriz por um número**

------------------------------------------------------------------------
