# 📖 Sobre o Projeto

Este repositório reúne exercícios de **lógica de programação** feitos
durante os estudos utilizando **Portugol Studio**.

O objetivo é desenvolver:

-   🧠 pensamento lógico
-   💻 base para programação
-   ⚙️ entendimento de algoritmos
-   📊 resolução de problemas

Os exercícios evoluem progressivamente desde **condições simples** até
**estruturas de dados como vetores e matrizes**.

# 🔀 Trabalho 01 --- Desvio Condicional

Exercícios:

-   Verificar se número é **par ou ímpar**
-   Verificar se pessoa é **maior de idade**
-   Comparar **dois números**
-   Verificar **aprovação de aluno**
-   Aplicar **desconto em compras**

Exemplo de lógica:

    se numero % 2 == 0
       escreva("Número par")
    senao
       escreva("Número ímpar")

------------------------------------------------------------------------

# 🔗 Trabalho 02 --- Operadores Lógicos

Exercícios:

-   Número entre **10 e 50**
-   Validação de **cadastro**
-   Verificar **prova final**
-   Número **não positivo**
-   **Login simples**

Exemplo:

    se idade >= 18 e documento == "S"
        escreva("Cadastro permitido")

------------------------------------------------------------------------

# 🔁 Trabalho 03 --- Laços de Repetição

Exercícios:

-   Contar de **1 até 10**
-   Somar números de **1 até N**
-   **Contagem regressiva**
-   **Tabuada**
-   Pedir senha **até acertar**

Exemplo:

    para i de 1 ate 10
        escreva(i)

------------------------------------------------------------------------

# ⚙️ Trabalho 04 --- Funções

Funções implementadas:

-   `somar(a,b)`
-   `quadrado(n)`
-   `ehPar(n)`
-   `calcularMedia(n1,n2,n3)`
-   `celsiusParaFahrenheit(c)`

Fórmula:

    F = (C * 1.8) + 32

------------------------------------------------------------------------

# 🔄 Trabalho 05 --- Recursividade

Problemas resolvidos:

-   **Fatorial**
-   **Soma de 1 até N**
-   **Fibonacci**
-   **Contagem regressiva**
-   **Inverter string**

Definição do fatorial:

    n! = n × (n-1)!

Casos base:

    0! = 1
    1! = 1

------------------------------------------------------------------------

# 📦 Trabalho 06 --- Vetores

Exercícios:

-   Armazenar números em vetor
-   Somar elementos
-   Encontrar **maior valor**
-   Contar **números pares**
-   **Inverter vetor**

Exemplo:

    Original: 1 2 3 4 5
    Invertido: 5 4 3 2 1

------------------------------------------------------------------------

# 🧮 Trabalho 07 --- Matrizes

Exercícios:

-   Criar e exibir **matriz**
-   Somar elementos
-   Encontrar maior valor
-   Somar **diagonal principal**
-   Multiplicar matriz por número

Diagonal principal:

    [0][0]
    [1][1]
    [2][2]
